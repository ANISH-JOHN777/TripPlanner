## Getting Started ğŸš€

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. **Clone or navigate to the project directory**
   ```bash
   cd FinalTrip
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   
   Copy `.env.example` to `.env`:
   ```bash
   cp .env.example .env
   ```
   
   Then edit `.env` and add your API keys:
   - **VITE_GEMINI_API_KEY**: Get from [Google AI Studio](https://makersuite.google.com/app/apikey)
   - **VITE_WEATHER_API_KEY**: Get from [OpenWeatherMap](https://openweathermap.org/api)

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   
   Navigate to `http://localhost:5173` (or the URL shown in your terminal)

## Usage ğŸ“–

### Planning a Trip

1. Click "Start Planning" or navigate to "Plan Trip"
2. Fill in your trip details:
   - Destination
   - Travel dates
   - Budget range
   - Number of travelers
   - Interests and preferences
3. Click "Generate Itinerary"
4. Review your AI-generated itinerary
5. Save the trip to view later

### Managing Trips

- View all saved trips in "My Trips"
- Filter trips by status (Planned, Ongoing, Completed)
- Search trips by destination
- Delete trips you no longer need

### Context Providers
